# MAPEK Coaching — Stone & Brass Edition

A polished, institution-level redesign of the MAPEK Coaching site.

## Palette

- **Background**: warm stone (`#E8E0D0` → `#EFE8D9` gradient with subtle paper grain)
- **Card surface**: lightest paper (`#F6F1E7`)
- **Primary ink**: deep espresso (`#2A241D` / `#3A332A`)
- **Accent**: brass (`#A8886A` / `#8A6F46`) for hairlines, eyebrows, and italic emphasis

## Typography

- **Display**: Cormorant Garamond (500/600, with italics for emphasis)
- **Body & UI**: Inter (300–600)

## Notes

- The dark logo was generated by mapping the original near-white pixels to deep espresso while preserving the teal accent.
- Reveal-on-scroll respects `prefers-reduced-motion`.
- Sticky header uses `backdrop-filter` for a soft frosted feel as you scroll past it.
- Fully responsive at 1024 / 880 / 480 breakpoints.
